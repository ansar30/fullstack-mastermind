## 🎯 How to Use This Repository

### For Interview Preparation
1. Start with the [30-Day Interview Prep Plan](./08-study-plans/30-day-interview-prep.md)
2. Focus on `interview-questions.md` in each topic
3. Practice [Coding Challenges](./07-interview-prep/coding-challenges/)
4. Review [System Design](./07-interview-prep/system-design/) patterns

### For Skill Development
1. Follow the [90-Day Mastery Roadmap](./08-study-plans/90-day-mastery-roadmap.md)
2. Read `overview.md` in each topic for detailed concepts
3. Study code examples in `code-examples/` folders
4. Use `cheatsheet.md` for quick reference

### For Quick Reference
- Use the `cheatsheet.md` files in each topic
- Bookmark frequently used commands and patterns
- Reference `resources.md` for additional learning materials
