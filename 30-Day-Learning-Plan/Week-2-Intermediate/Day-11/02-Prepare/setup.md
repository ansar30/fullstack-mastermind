# Day 11 - Setup Guide

## Prerequisites

- Node.js project from Day 05
- Express installed
- Understanding of async/await

## Setup

Continue with existing Express project:
```bash
cd day05-node-app
# Or create new
mkdir day11-node-app
cd day11-node-app
npm init -y
npm install express
```

## Files to Create

- `middleware/errorHandler.js` - Error middleware
- `utils/asyncHandler.js` - Async wrapper utility
- `recursion-problems.js` - Recursion practice

## Ready to Start!

Move to `03-Solve` folder to begin coding.
