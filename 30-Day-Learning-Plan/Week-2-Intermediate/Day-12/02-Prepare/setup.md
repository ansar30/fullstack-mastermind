# Day 12 - Setup Guide

## Prerequisites

- MongoDB installed or Atlas account
- Mongoose installed
- Express project from Day 06

## Setup

Continue with existing MongoDB project:
```bash
cd day06-api
# Ensure mongoose is installed
npm install mongoose
```

## MongoDB Setup

### Local MongoDB
```bash
mongod
```

### MongoDB Atlas
Use connection string from Atlas dashboard

## Files to Create

- `models/User.js` - User model with indexes
- `aggregation-queries.js` - Aggregation examples
- `prefix-sums.js` - Prefix sum problems

## Ready to Start!

Move to `03-Solve` folder to begin coding.
