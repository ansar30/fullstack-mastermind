# Day 08 - Setup Guide

## Prerequisites

- React project from previous days
- React DevTools browser extension
- Performance profiling tools

## Setup

Continue with existing React project or create new:
```bash
npm create vite@latest day08-react-app -- --template react
cd day08-react-app
npm install
```

## Performance Tools

- React DevTools Profiler
- Chrome Performance tab
- React.memo for optimization

## Files to Create

- `VirtualizedList.jsx` - Virtualized list component
- `sliding-window-medium.js` - Medium sliding window problems

## Ready to Start!

Move to `03-Solve` folder to begin coding.
