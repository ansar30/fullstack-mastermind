# Day 09 - Setup Guide

## Prerequisites

- React project from previous days
- Understanding of useState and useEffect

## Setup

Continue with existing React project:
```bash
cd day08-react-app
# Or create new
npm create vite@latest day09-react-app -- --template react
cd day09-react-app
npm install
```

## Files to Create

- `AppContext.jsx` - Context setup
- `AppProvider.jsx` - Provider component
- `hashmap-set-problems.js` - Advanced HashMap/Set problems

## Ready to Start!

Move to `03-Solve` folder to begin coding.
