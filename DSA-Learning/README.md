# DSA Learning Journey - 15 Days

Welcome to your Data Structures and Algorithms learning plan! This is a structured 15-day program to master fundamental DSA concepts.

## 📚 Learning Path

### Week 1: Foundations
- **Day 1-3**: Arrays (Two pointers, Sorting, Frequency map)
- **Day 4-6**: Strings (Reverse, Anagram, Compression)
- **Day 7-9**: Hash Maps & Sets (Occurrences, Duplicates, Grouping)

### Week 2: Advanced Patterns
- **Day 10-12**: Sliding Window (Longest substring, Subarray sum, Max sum)
- **Day 13-15**: Trees & Recursion (Traversal, Flattening, Sum)

## 📂 Structure

Each day folder contains:
- `README.md` - Day's topics and problem list
- `problems/` - Individual problem files with templates
- `solutions/` - Your solved implementations
- `notes.md` - Learnings and key concepts

## ✅ Daily Workflow

1. Read the day's `README.md`
2. Understand the concepts in `notes.md`
3. Solve problems in `problems/` folder
4. Move completed solutions to `solutions/`
5. Review and optimize your code

## 🎯 Goal

Complete all problems with clean, optimized solutions and deep understanding of the underlying concepts.
