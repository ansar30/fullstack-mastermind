## Two Pointer Technique

### Concept
Two pointers is an efficient technique for solving array/string problems by using two references that move through the data structure.

### Common Patterns

#### 1. Opposite Direction (Left & Right)
```
[1, 2, 3, 4, 5]
 ↑           ↑
left       right
```
- Used for: Palindrome check, Two Sum in sorted array, Container with most water

#### 2. Same Direction (Slow & Fast)
```
[1, 1, 2, 2, 3]
 ↑  ↑
slow fast
```
- Used for: Remove duplicates, Move zeros, Partition array

### Time Complexity
- Usually O(n) - single pass through array
- Better than nested loops O(n²)

### Space Complexity
- Usually O(1) - only pointer variables
